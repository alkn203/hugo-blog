002-Shapeの位置指定

Oct 3, 2016   #tips  #shape 

Table of contents

    • 内容
    • コード
    • コード説明
    • 他の位置指定方法

内容

Shapeの位置を以下のように指定します。

コード

// グローバルに展開
phina.globalize();
/*
 * メインシーン
 */
phina.define("MainScene", {
  // 継承
  superClass: 'DisplayScene',
  // 初期化
  init: function() {
    // 親クラス初期化
    this.superInit();
    // 背景色
    this.backgroundColor = 'black';
    // Shapeを作成してシーンに追加
    var shape = Shape().addChildTo(this);
    // 移動
    shape.x = 320;
    shape.y = 480;
  },
});
/*
 * メイン処理
 */
phina.main(function() {
  // アプリケーションを生成
  var app = GameApp({
    // MainScene から開始
    startLabel: 'main',
  });
  // fps表示
  //app.enableStats();
  // 実行
  app.run();
});

[runstantで実行]

コード説明

  • Shapeの位置変更は、x, yプロパティに直接数値を指定することで可能です。

他の位置指定方法

  • setPosition関数を使えば、x, yの値を一括で指定することができ、生成から一気にチェインメソッドで繋げて書くこともできるので便利です。
var shape = Shape().addChildTo(this).shape.setPosition(320, 480);

[runstantで実行]

  • Shapeのコンストラクタにパラメータとして与えてることでも指定できます。
var shape = Shape({
  x: 320,
  y: 480
}).addChildTo(this)

[runstantで実行]

  • moveBy関数を使えば、x, yの移動量で位置を変更することができます。
shape.setPosition(320, 480).moveBy(100, 200);

[runstantで実行]

  • 変則的ですが、Vector2クラスを使ってベクトル値の加算で位置指定する方法もあります。
var v = Vector2(100, 200);
shape.position.add(v);

[runstantで実行]
